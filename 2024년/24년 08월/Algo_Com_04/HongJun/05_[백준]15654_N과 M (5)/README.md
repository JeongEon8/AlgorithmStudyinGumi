# [백준 S3] 15651. N과 M (5)
 
## ⏰  **소요 시간**
20분

## :pushpin: **Algorithm**
순열, 백트래킹

## ⏲️**Time Complexity (시간 복잡도)**
$O(N)$

## :round_pushpin: **Logic**
- 처음에 백트래킹으로 푸려고 하다가 중복된 수가 추가되더라.
- 그래서 순열로 풀릴 것 같아서 permutaions를 사용했다.
- 바로 풀리더라.

## :black_nib: **Review**
- 중복되는 수열이 안되면 permutaions를 사용할 수 있다.
