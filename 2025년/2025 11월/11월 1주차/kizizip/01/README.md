# [백준 - S4] 24431. 유사 라임 게임


## ⏰ **time**
30분

## :pushpin: **Algorithm**
문자열, 자료 구조 

## ⏲️**Time Complexity**
O(t*n)

## :round_pushpin: **Logic**
1. 모든 단어를 순회하며 마지막 f글자(접미사)를 추출
2. 접미사가 집합에 없는 경우, 아직 짝이 없는 접미사로 판단해 집합에 추가
3. 이미 집합에 존재하는 경우, 동일한 접미사를 가진 단어가 두 개 생겼으므로 한 쌍을 완성
  - 이때 count를 1 증가시키고,
  - 형성된 쌍이 다시 중복 계산되지 않도록 해당 접미사를 집합에서 제거한다.
4. 모든 단어를 처리하면 count에는 형성 가능한 유사 라임 쌍의 총 개수가 남는다.
