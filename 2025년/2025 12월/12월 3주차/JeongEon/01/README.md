# [백준 - 실버 4] 1748. 수 이어 쓰기 1

## ⏰**time**
10분

## :pushpin: **Algorithm**

수학, 구현

## ⏲️**Time Complexity**

$O(\log N)$

* 자릿수 단위(1, 10, 100, …)로 반복하며 계산

## :round_pushpin: **Logic**

1. 1부터 N까지의 수를 이어서 쓸 때, 각 수의 **자릿수**가 중요하다.
2. 자릿수가 같은 구간별로 나누어 계산한다.

   * 1자리 수: 1 ~ 9
   * 2자리 수: 10 ~ 99
   * 3자리 수: 100 ~ 999 …
3. `i`를 자릿수의 시작 값(1, 10, 100, …)으로 두고,

   * 해당 자릿수 구간에서 등장하는 숫자의 개수는 `(N - i + 1)`
4. 이를 모두 더하면 전체 자릿수의 합이 된다.

```
cnt = 0
for i = 1; i <= N; i *= 10:
    cnt += (N - i + 1)
```

* 예시: N = 120

  * 1자리: 120 - 1 + 1 = 120
  * 2자리: 120 - 10 + 1 = 111
  * 3자리: 120 - 100 + 1 = 21
    → 총 자릿수 = 252

## :black_nib: **Review**
* 단순히 1부터 N까지 이어 붙이면 시간 초과가 나지만,
  **자릿수 규칙**을 이용하면 매우 깔끔하게 해결 가능했다.
* 수학적 관찰이 중요한 문제라는 걸 다시 느꼈다.
