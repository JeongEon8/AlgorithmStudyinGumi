# [백준- S3] 2346. 풍선 터뜨리기
 
## ⏰  **time**
30분

## :pushpin: **Algorithm**
- 덱

## ⏲️**Time Complexity**
$O(N)$

## :round_pushpin: **Logic**
- 1번 풍선부터 터뜨리기 시작해서 해당 풍선의 번호만큼 이동함
- 양수는 오른쪽, 음수는 왼쪽으로 이동해서 터뜨림
- 이미 터진 풍선은 뺴고 이동함

1. 풍선 리스트에서 첫 번째 숫자를 빼서 저장한다.
2. 리스트가 빌 때까지 반복문 안에 풍선 번호가 음수/양수인 경우를 고려해서 Index를 계산한다.
3. 기존 번호에서 풍선 번호를 더한 수 % 리스트 길이
4. 인덱스에 해당하는 숫자들을 계속 pop해서 출력한다.
