# [백준 - 실버 1] 15990. 1, 2, 3 더하기 5

## ⏰ **time**
33분

## :pushpin: **Algorithm**
DP

## ⏲️ **Time Complexity**
$O(N)$

## :round_pushpin: **Logic**
1. 마지막으로 오는 수가 1이면? 그 앞에 올 수 있는 숫자는 2 또는 3이 된다.
2. 마지막으로 오는 수가 2이면? 그 앞에 올 수 있는 숫자는 1 또는 3이 된다.
3. 마지막으로 오는 수가 3이면? 그 앞에 올 수 있는 숫자는 1 또는 2이 된다.
4. 따라서 점화식도 2차원 방정식으로 다음과 같다.
   > D[n][1] = D[n-1][2] + D[n-1][3]
   > 
   > D[n][2] = D[n-2][1] + D[n-2][3]
   > 
   > D[n][3] = D[n-3][1] + D[n-3][2]
   >
   > -> result = D[n][1] + D[n][2] + D[n][3]

```cpp
	for (int i = 4; i <= 1000000; i++) {
		if (i - 1 >= 0) {
			DP[i][1] = (DP[i - 1][2] + DP[i - 1][3]) % 1000000009;
		}
		if (i - 2 >= 0) {
			DP[i][2] = (DP[i - 2][1] + DP[i - 2][3]) % 1000000009;
		}
		if (i - 3 >= 0) {
			DP[i][3] = (DP[i - 3][1] + DP[i - 3][2]) % 1000000009;
		}
	}
```
