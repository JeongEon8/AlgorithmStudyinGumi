# [백준 - S4] 1269. 대칭 차집합

## ⏰ **time**

15분

## :pushpin: **Algorithm**
- 자료구조
- 해시를 사용한 집합과 맵

## ⏲️**Time Complexity**

$O(N)$

## :round_pushpin: **Logic**
- 두 집합을 set으로 입력받는다
- 파이썬에서 차집합은 set 자료형을 서로 빼주면 된다
- 서로 겹치는 숫자를 빼고 남은 숫자의 개수를 더하면 답
