# [PGS - Lv2] 정수 제곱근 판별

## ⏰**time**

3분

## :pushpin: **Algorithm**


## ⏲️**Time Complexity**

$O(1)$

## :round_pushpin: **Logic**

1. `Math.sqrt(n)`으로 n의 제곱근을 구하여 long변수 x에 담는다
2. x의 제곱이 n과 일치하면 x+1의 제곱 리턴
3. 그렇지 아니하다면 -1 리턴
