# [백준 - 골드 5] 1074.Z
 
## ⏰  **time**
2시간

## :pushpin: **Algorithm**
재귀

## ⏲️**Time Complexity**
$O(N)$

## :round_pushpin: **Logic**
1. 일단 계속 n을 1 빼기 -> n=0이 되면 1*1로 끝
2. 1사분면, 2사분면, 3사분면, 4사분면 중 어디에 포함되어있는지 찾기
3. 재귀로 넘길 때 한 사분면의 (0,0)의 방문 순서 같이 넘기기
4. (0,0)의 방문 순서는 1사분면은 0, 2사분면은 (2^N*2^N)/4, 3사분면은 2^N/4*2, 4사분면은 2^N/4*3
5. 기존 방문 순서 + 새로운 사분면의 방문 순서
