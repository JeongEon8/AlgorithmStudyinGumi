# [PGS - Lv1] 01_두 정수 사이의 합

## ⏰**time**
3분

## :pushpin: **Algorithm**
수학

## ⏲️**Time Complexity**
$O(N)$

## :round_pushpin: **Logic**
1. a와 b 중 큰 수를 big, 작은 수를 small에 저장
2. small부터 big까지 합
   ```cpp
    for(int i = small; i <= big; i++){
        answer += i;
    }
   ```
