# [백준 - S5] 1769. 3의 배수


## ⏰ **time**
20분

## :pushpin: **Algorithm**
문자열, 구현

## ⏲️**Time Complexity**
O(N)

## :round_pushpin: **Logic**
- 입력한 수의 각 자리수들을 더한 값 Y를 만든다.
- 이 과정을 Y가 한자리 수가 될 때 까지 반복한다.
- 그 뒤에 Y가 3의 배수이면 YES, 아니면 NO를 출력한다.
