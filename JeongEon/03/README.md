# [백준 - 실버 4] 9372. 상근이의 여행

## ⏰**time**

10분

## :pushpin: **Algorithm**

그래프, 트리, MST(최소 신장 트리)

## ⏲️**Time Complexity**

$O(M)$
(입력 간선 개수만큼 순회합니다. MST 성질을 사용해 정답을 바로 도출함.)

## :round_pushpin: **Logic**

1. 입력으로 주어진 비행 스케줄은 무방향 그래프의 간선 정보입니다.
2. 이 문제는 **최소한의 비행으로 모든 나라를 방문하는 경우의 수**를 묻는 문제이며,
3. 사실상 그래프에서 **모든 노드를 연결하는 최소 간선 개수**, 즉 **트리의 간선 수는 (노드 수 - 1)**이라는 MST의 성질을 활용하면 됩니다.

```
모든 테스트 케이스에 대해 나라 수 N과 비행기 수 M을 입력받고,
입력으로 주어진 M개의 간선 정보를 단순히 무시한 채
각 테스트 케이스에서 N - 1을 출력하면 됩니다.
```

## :black_nib: **Review**

* 입력만 받으면 바로 MST의 성질을 적용해 `N - 1`을 출력하면 되는 쉬운 문제였습니다.
* 처음에 DFS나 BFS를 고민할 수 있지만, 그래프가 연결된다는 조건 하에 MST의 기본 성질을 바로 적용하는 것이 핵심입니다.
