# [백준 - 실버 2] 15888. 정답은 이수근이야!!

## ⏰**time**

20분

## :pushpin: **Algorithm**

수학, 완전 탐색(브루트포스)

## ⏲️**Time Complexity**

$O(K^2)$

* 제한된 범위 내에서 두 해를 탐색
* 상수 범위 반복(최대 약 (400 \times 400))이므로 충분히 빠름

## :round_pushpin: **Logic**

1. 주어진 이차방정식
   [
   ax^2 + bx + c = 0
   ]
   에 대해 **서로 다른 두 정수 해**가 존재하는지 판별하는 문제.
2. 먼저 판별식을 계산:

   * (b^2 - 4ac \le 0) 이면 서로 다른 두 해가 존재할 수 없으므로 `"둘다틀렸근"` 출력.
3. 문제 조건에 따라 두 가지 케이스를 검사:

   * **이수근**: 두 해가 모두 **2의 거듭제곱(2, 4, 8, …, 256)** 인 경우
   * **정수근**: 두 해가 **서로 다른 정수(-200 ~ 200)** 인 경우
4. 각 범위에서 서로 다른 두 값 `i`, `j`에 대해 방정식을 만족하는지 브루트포스로 검사.
5. 우선순위:

   * 이수근 조건을 만족하면 `"이수근"`
   * 아니면 정수근 조건을 만족하면 `"정수근"`
   * 둘 다 아니면 `"둘다틀렸근"`

```
판별식 <= 0 → 둘다틀렸근

for i, j in (2의 거듭제곱 범위):
    if i != j and f(i) == 0 and f(j) == 0:
        이수근

for i, j in (-200 ~ 200):
    if i != j and f(i) == 0 and f(j) == 0:
        정수근

else:
    둘다틀렸근
```

## :black_nib: **Review**
* 수학 문제처럼 보이지만, 실제로는 **조건이 제한된 완전 탐색 문제**였다.
* 판별식을 먼저 체크해 불필요한 계산을 줄인 점이 인상적.
* 문제 이름과 출력 문자열이 독특해서 기억에 남는 문제였다.
