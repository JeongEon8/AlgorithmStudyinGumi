## :round_pushpin: **Logic**
1. 토너먼트에서 각 라운드마다 참가자 번호는
   `현재 번호 / 2 (올림)` 형태로 갱신된다.
2. 두 선수가 **같은 라운드에서 만나려면**
   * 번호 차이가 1이고
   * 더 큰 번호가 짝수여야 한다.
3. 위 조건을 만족할 때까지 라운드를 반복하며,
   매 라운드마다 김지민과 임한수의 번호를 갱신한다.
4. 조건을 만족하는 순간의 라운드 번호를 출력한다.

```
while (두 선수가 아직 만나지 않음):
    kim = (kim + 1) / 2
    im  = (im + 1) / 2
    round++
```
