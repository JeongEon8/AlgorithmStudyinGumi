# [PGS-LV1] 약수의 합

## ⏰ **time**

5분

## :pushpin: **Algorithm**
완전탐색

## ⏲️ **Time Complexity**
$O(N)$

## :round_pushpin: **Logic**

1. 정수 n의 약수는 n을 i로 나눴을 때 나머지가 0인 수
2. 반복문을 1부터 n/2까지 돌면서 약수를 찾는다
3. 약수인 경우 answer에 누적
4. 마지막에 n 자신도 약수이므로 더해준다
