# [SWEA - D4] 1219. [S/W 문제해결 기본] 4일차 - 길찾기

## ⏰  **time**
25분

## :pushpin: **Algorithm**
배열 + bfs

## ⏲️**Time Complexity**
$O(N^2)$

## :round_pushpin: **Logic**
1. index에 start 지점을 넣고, 값으로 end에 해당하는 지점들의 리스트를 넣는다.
2. 리스트에 들어있는 값들을 반복문을 통해 방문할 수 있는 경우 queue에 추가
3. 방문예정인 지점이 99인 경우 도달할 수 있는 경우이므로 1 반환하고, 바로 끝낼 수 있도록 queue를 clear한다.
