# [백준 - S5] 2535. 아시아 정보올림피아드

## ⏰ **time**

10분

## :pushpin: **Algorithm**

- 수학

## ⏲️**Time Complexity**

$O(1)$

## :round_pushpin: **Logic**
가장 밑에 돌을 꺼내기 위해선 그 위에 모든 돌을 치워야한다.  
즉 치우는데 n-1번 옮기고 가장 아래 돌을 옮기는데 1번해서 하나를 옮길때마다 남은 돌의 갯수만큼 움직여야한다.  
1~n까지의 합이 답이다.
