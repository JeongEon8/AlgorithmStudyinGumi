# [백준 - S5] 33664. 현실적인 생일 축하 방안


## ⏰ **time**
5분

## :pushpin: **Algorithm**
해시를 사용한 집합과 맵

## ⏲️**Time Complexity**
O(n+m)

## :round_pushpin: **Logic**
1. 해시맵에 아이템 이름(키값), 가격을 저장한다.
2. 사려는 아이템을 키값으로서, 가격 총합을 조회한다.
3. 예산을 넘는다면 `unacceptable` 출력 후 리턴
4. 끝까지 도달하면 `acceptable` 출력
