## :round_pushpin: **Logic**
1. 일단 stack에 첫글자부터 넣고 시작한다.
2. 스택의 마지막 저장된 문자와, 그 다음 문자가 일치하는지 확인하고
3. 일치한다면 pop, 아니라면 push
4. 마지막에 stack이 깔끔하게 비었는지 기준으로 출력
