# [백준] 9372. 상근이의 여행 


## ⏰ **time**
20분

## :pushpin: **Algorithm**
그래프 이론, 트리 

## ⏲️**Time Complexity**
O(t+m)

## :round_pushpin: **Logic**
항상 정점 개수(n) - 1을 출력하는 이유? <br/>
모든 노드 연결하는 최소 간선 개수 = n-1 <br/>
즉, 트리 구조 이루려면 간선이 최소 n-1개 필요하므로... 간선 정보와 관계없이 항상 n-1 출력
