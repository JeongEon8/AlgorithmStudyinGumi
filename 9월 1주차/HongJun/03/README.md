# [백준 - S4] 1269. 대칭 차집합
 
## ⏰  **time**
20분

## :pushpin: **Algorithm**
- 집합, 맵

## ⏲️**Time Complexity**
$O(N)$

## :round_pushpin: **Logic**
- 파이썬의 set 자료형을 사용해서 집합의 원소들을 넣는다.
- set에서 차집합을 구하려면 그냥 빼기 연산을 해주면 된다.
- a-b, b-a를 하고 각 원소의 개수를 구해서 더한다.
