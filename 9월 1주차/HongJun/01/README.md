# [백준 - S5] 29615. 알파빌과 베타빌
 
## ⏰  **time**
30분

## :pushpin: **Algorithm**
- 그리디

## ⏲️**Time Complexity**
$O(N)$

## :round_pushpin: **Logic**
- 3번 째 줄에서 입력받은 대기 번호를 인덱스로 해서 대기 인원 중에 해당 인덱스의 숫자를 찾는다.
- 해당 숫자를 따로 리스트에 저장한 뒤, 정렬을 한다.
- 친구가 인덱스를 저장한 값 안에 들어있지 않으면 +1을 카운트 한다.
