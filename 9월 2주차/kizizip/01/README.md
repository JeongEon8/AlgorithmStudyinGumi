# [프로그래머스 - 2] JadenCase 문자열 만들기


## ⏰ **time**
10분

## :pushpin: **Algorithm**


## ⏲️**Time Complexity**
O(n)

## :round_pushpin: **Logic**
1. 문자열을 char 배열로 변환
2. 첫 글자가 소문자면 대문자로 변환
3. 두 번째 문자부터 끝까지 순회하면서
- 이전 문자가 공백이고 현재 문자가 소문자면 대문자로 변환
- 공백이 아니고, 현재 문자가 대문자면 소문자로 변환
4. 변환된 char 배열을 문자열로 만들어 반환
