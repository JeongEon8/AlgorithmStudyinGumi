# [백준 - G4] 4485. 녹색 옷 입은 애가 젤다지? 

## ⏰ **time**
40분

## :pushpin: **Algorithm**
- 최단거리, 다익스트라

## ⏲️**Time Complexity**
$O(v+e)$

## :round_pushpin: **Logic**
- heapq를 사용해서 우선순위 큐로 최소값을 먼저 꺼낸다.
- 최소 비용을 계산하는 2차원 배열을 무한대 값으로 초기화한다.
- 우선순위 큐를 돌면서 도착 위치 (N-1, N-1)에 도달하면 종료한다.
- 4방향으로 이동하면서 (현재까지 온 비용 + 다음 칸 비용) < (기존에 기록된 다음 칸 비용)인지 확인한다.
- 최소비용 테이블을 갱신하고, 우선순위 큐에 (비용, 좌표)를 넣는다.
