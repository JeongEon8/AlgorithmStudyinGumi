# [백준 S1] 5525. IOIOI
 
## ⏰  **소요 시간**
30분

## :pushpin: **Algorithm**
문자열

## ⏲️**Time Complexity (시간 복잡도)**
처음 시도했을 때: $O(N*N)$
정답 코드: $O(N*logN)$

## :round_pushpin: **Logic**
- 처음에 문자열 슬라이싱으로 풀었는데 50점만 맞았다.
- 시간복잡도가 N^2이 되기 때문에 나머지는 시간초과가 걸린다.
- 그래서 처음에 IOI로 시작해서 N번 만큼 OI를 더해서 확인한다.
