# [백준- S2] 18870. 좌표 압축
 
## ⏰  **time**
40분

## :pushpin: **Algorithm**
- 정렬, 좌표압축

## ⏲️**Time Complexity**
$O(N)$

## :round_pushpin: **Logic**
- 입력받은 리스트를 작은 수 부터 중복없이 정렬한다.
- 그리고 딕셔너리에 정렬된 리스트의 값을 key로 하고 해당 숫자를 value로 넣는다
- 그 딕셔너리를 출력하면 된다.
